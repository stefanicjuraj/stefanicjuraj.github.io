A non-exhaustive list of CLI commands and tools I use.

## Commands

### System and OS

```bash
uname -a          # kernel and architecture
uname -m          # CPU architecture
hostname          # system hostname
uptime            # system run time and load
date              # current system time
whoami            # current user
```

### Processes and execution

```bash
ps aux            # all processes
ps -ef            # full process list
top               # live process view
pgrep <name>      # process id by name
lsof -p <pid>     # files used by process
```

### File System and Disk

```bash
pwd               # current directory
ls -lh            # files with sizes
df -h             # disk usage by mount
du -sh <path>     # directory size
mount             # mounted filesystems
```
