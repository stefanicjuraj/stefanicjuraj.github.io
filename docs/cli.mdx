A non-exhaustive list of command line tools I use.

## Tools

| Tool                                                                | Description                                                                                                                    |
| :------------------------------------------------------------------ | :----------------------------------------------------------------------------------------------------------------------------- |
| [sysview](https://github.com/stefanicjuraj/sysview)                 | command-line tool to list active network processes and listening ports                                                         |
| [todo.md](https://github.com/stefanicjuraj/todo.md)                 | local, simple, and fast command-line todo application with export to markdown                                                  |
| [changelog.world](https://github.com/stefanicjuraj/changelog.world) | changelogs and release notes in developer-friendly format                                                                      |
| [vale](https://vale.sh/)                                            | markup-aware linter for prose built                                                                                            |
| [docs2llms](https://github.com/stefanicjuraj/docs2llms)             | consistent text formatting for AI and LLMs by extracting documentation and converting it into llms.txt and llms-full.txt files |
| [docsee](https://github.com/stefanicjuraj/docsee)                   | analyze repositories and generate an interactive report with a file tree and repository insights                               |
| [wttr.in](https://github.com/chubin/wttr.in)                        | console-oriented weather forecast service                                                                                      |
